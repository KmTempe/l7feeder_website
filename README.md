## Tech Stack

- **React** & **Vite**
- **Material-UI**
- **Vercel** for hosting

## Getting Started

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/KmTempe/portfolio-react.git
    cd portfolio-react
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run locally:**
    ```bash
    npm run dev
    ```

## Deployment

The site is automatically deployed to Vercel whenever changes are pushed to the `master` branch.
